# TnonMusic

TonoMusic是一个基于flutter的LxMusic插件兼容项目，其通过flutter_js项目实现了一个最小的LxMusic插件运行时。

# Q&A

Q:为什么在LXmusic中可以正常工作的插件无法运行  
A:项目仍处在开发中，目前仅实现了最小的兼容。已知处于被混淆状态的脚本会出现严重的运行问题。
